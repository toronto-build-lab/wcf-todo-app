# Todo SPA

Modern single-page Todo application scaffolded with Vite, React, and TypeScript.

## Phase 1 Scope

- Local-only task management (create, edit, delete, toggle complete)
- Browser localStorage persistence
- Light and dark theme toggle
- No backend connectivity

## Requirements

- Node.js 20.19+ or 22.12+
- pnpm

If pnpm is not installed yet:

```bash
corepack enable
corepack prepare pnpm@latest --activate
```

## Run Locally

```bash
pnpm install
pnpm run dev
```

## Commands

```bash
pnpm run build
pnpm run lint
pnpm run test
pnpm run preview
```
